**Backend**
**Description**
This is a backend project built with Node.js and Express. The project provides a robust framework for building web applications and APIs. It includes various essential dependencies and development tools to help you get started quickly.

**Requirements**
To run this project, you'll need to have Node.js and npm (Node Package Manager) installed on your system. You can download and install Node.js from nodejs.org.


**Dependencies**
Install the dependencies,The project relies on the following dependencies:
cors: ^2.8.5
dotenv: ^16.4.5
express: ^4.18.2
mongoose: ^8.2.0
ws: ^8.16.0 

**Scripts**
The following npm scripts are available in the project:

start: Runs the application using Node.js.
npm start


**Environment Variables**
This project uses the `dotenv` package for environment variables. Create a `.env` file in the root directory and add your variables there.
PORT=your_port_number
URI=your_database_uri

I have created a database in `MongoDB` , which stores all the data which we enter in the frontend.


**Usage**
Once the server is running, you can interact with the API using tools like Postman or via your frontend application. Make sure to update the database URI and any other necessary environment variables before running the application.
